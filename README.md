# Shardwake

> Mobile 2.5D stylized fantasy extraction RPG.

Shardwake is a mobile-first PvPvE extraction RPG set in a shattered fantasy world.

Players enter unstable magical fragments called **Shards**, fight monsters and other Seekers, collect relics, and escape through portals before the world collapses.

The game has no fixed character classes.

Instead, players build their playstyle through weapons, armor weight, stats, loot and skills.

## Core Identity

- Mobile 2.5D fantasy extraction RPG
- Weapon-based builds, no fixed classes
- PvPvE combat
- Procedural map layout from handcrafted biome modules
- Armor weight system
- Hellgate overtime phase
- Loot risk and full-loss death
- AI-generated run chronicles and item stories

## Current Status

Prototype / pre-production.

The current goal is not to build the full game immediately.

The current goal is to create a small playable vertical slice and validate whether the core loop is fun:

Enter Shard → fight → loot → survive → extract → improve build → repeat.

## Main Design Rule

Every system must create meaningful player choice.

No pay-to-win.
No autoplay.
No giant files.
No hardcoded balance inside UI.
