# Shardwake — Haven Hub MVP

Haven is the player's hub between extraction runs.

For MVP, Haven is a generated UI screen, not a full 3D city.

## Buildings

### Expedition Gate

The main start point.

Player can:
- choose weapon build preset
- choose armor load preset
- enter the Shard

### Stash

Stores loot successfully extracted from runs.

Player can:
- preview stored items
- equip first stored items for greybox testing

### Merchant

Early economy testing.

Player can:
- sell the first stash item for Shard Dust
- buy a starter Healing Flask

### Tavern

Non-AI rumor system for MVP.

Player can:
- read random tips and world rumors
- learn about portals, mini-bosses, Hellgates and risk

### Job Board

Contract system placeholder.

Player can:
- accept a simple greybox contract
- gain Shard Dust reward

## Design Goal

Haven should become the safe planning layer:

prepare build → enter run → extract loot → return to Haven → improve build.
